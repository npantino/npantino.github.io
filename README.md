# npantino.github.io
A simulator of the Casio FX-260 Solar II scientific calculator. Made for students at UC Davis taking the PHY 9 Series, to practice using these calculators for exams. <br>
Note: The x^1/y operation still doesn't work, it acts like x^y. Also, there are plenty of bugs still, so just hit the AC button if something goes wrong. <br>
This is my first time making a website, so any tips on making the user interface look better is appreciated. <br>
